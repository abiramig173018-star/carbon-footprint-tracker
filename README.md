# 🌍 CarbonTrace

> Know your impact. Own your actions.

A personal carbon footprint calculator built in ~45 minutes 
using vibe coding for **Google PromptWars 2026**.

## Live Demo
[▶ Try it here](https://carbon-footprint-cal-ivjt.bolt.host)

## What It Does
- 5-question lifestyle quiz (transport, flights, diet, energy, shopping)
- Instant CO₂ score with A–F grade vs global average (4.7t/yr)
- Visual category breakdown
- Personalized action plan with live savings estimator

## Built With
- React (single component, no framework)
- Pure CSS — zero chart libraries
- Real CO₂ data from Our World in Data, IEA, ICAO

## Run Locally
git clone https://github.com/yourusername/carbon-footprint-tracker
cd carbon-footprint-tracker
npm install && npm start

## Data Sources
| Category | Source |
|----------|--------|
| Transport | Our World in Data, 2023 |
| Diet | Poore & Nemecek, Science (2018) |
| Flights | ICAO Carbon Emissions Calculator |
| Energy | IEA Residential Report, 2024 |
| Shopping | Carbon Trust Study |
